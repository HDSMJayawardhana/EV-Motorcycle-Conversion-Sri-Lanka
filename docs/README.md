## 📌 Key Findings Summary

- **Prototype:** Bajaj CT100 (most popular motorcycle in Sri Lanka)
- **Motor:** 1000W, 48V BLDC (Model: MY1020)
- **Battery:** 48V, 40Ah Li-ion (13S15P configuration)
- **Estimated Range:** ~50 km per full charge
- **Payback Period:** ~2 years (LKR 7,500/month savings)
- **Weight Distribution:** Front 46.6% | Rear 53.4% ✅ Safe
